# AI Assistant Context Files

This directory contains documentation to help AI assistants understand and work with the Agents & Arbiters codebase.

## Files

- **`overview.md`** - Project description, current status, what works, what's next
- **`development.md`** - Setup instructions, commands, environment config, coding standards
- **`architecture.md`** - Layered architecture patterns, domain objects, frontend patterns
- **`azure-deployment.md`** - Azure deployment guide with infrastructure and configuration

## Quick Start for AI Assistants

1. Read `overview.md` first to understand what this project is
2. Check `architecture.md` to understand code organization and patterns
3. Refer to `development.md` for setup and coding standards
4. Use `azure-deployment.md` when working on Azure deployment tasks
